# Miss Google
<p>此系統主要幫助創作者簡單使用 Google 語音於影片配音中。</p>
<p>系統特點如下</p>
<ul>
  <li>完全免費</li>
  <li>可調整語速</li>
  <li>可選擇輸出資料夾</li>
</ul>

# 安裝使用
```
npm i -D electron@latest
npm i
```

# 執行
```
npm run start 
```

# 打包 Windows
```
npm run build_win 
```
